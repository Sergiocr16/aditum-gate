Tipo de equipo

Define qué módulos se muestran abajo. La configuración técnica (scannerType, pantalla…) se arma sola según el tipo.

Identidad

El id físico del equipo (device-id.txt) manda: si acá se indica otro distinto, el guardado se rechaza (409). Vacío = config genérica.
Condominio o sitio donde está instalado el equipo.
Tipo de código QR que acepta el equipo. Usá Seguro salvo en sitios con códigos viejos. El servidor es siempre caseta.aditumcr.com.

Lectores

Cómo lee los QR este equipo: lector pistola o cámara USB.

Hasta 2 lectores. role: entrada (entry) o salida (exit) · doorId: id de la puerta que abre en Aditum. Según el tipo: device name (pistola) o cámara (índice USB).

Este tipo de equipo no usa lectores pistola.
